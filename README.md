# Favorita Grocery Sales Forecasting

## Описание

Решение задачи прогнозирования продаж для соревнования [Corporación Favorita Grocery Sales Forecasting](https://www.kaggle.com/competitions/favorita-grocery-sales-forecasting) на Kaggle.

**Метрика**: NWRMSLE (Normalized Weighted Root Mean Squared Logarithmic Error)

## Данные

Датасет расположен по пути `/kaggle/input/datasets/nikolasking/favorita/`.

Файлы:
| Файл | Описание |
|------|----------|
| `train.csv` | Продажи по дате, магазину, товару (2013–2017) |
| `test.csv` | 16-дневный тестовый горизонт (авг 2017) |
| `stores.csv` | Метаданные магазинов (город, штат, тип, кластер) |
| `items.csv` | Метаданные товаров (семейство, класс, скоропортящийся) |
| `transactions.csv` | Количество транзакций по дате и магазину |
| `oil.csv` | Цены на нефть (экзогенный признак) |
| `holidays_events.csv` | Праздники и события в Эквадоре |

## Описание подхода

### Постановка задачи
Целью работы является прогнозирование ежедневных продаж потребительских товаров в сети супермаркетов Corporación Favorita (Эквадор) на 16 дней вперед (с 16 по 31 августа 2017 года). Задача классифицируется как многомерное прогнозирование временных рядов (Time Series Forecasting) в условиях высокой размерности: данные включают более 4000 уникальных товаров, распределенных по 54 магазинам. Основная сложность заключается в необходимости учета внешних факторов, таких как динамические цены на нефть (влияющие на экономику Эквадора), государственные праздники, локальные промо-акции и специфику категорий товаров. Оценка качества производится по метрике NWRMSLE, которая накладывает повышенный штраф за ошибки в прогнозе скоропортящихся продуктов, что требует от моделей не только фиксации общих трендов, но и точного учета жизненного цикла различных групп товаров.

### Протокол валидации
Time-based split: последние 16 дней обучающей выборки (01–15 авг 2017) используются как локальная валидация, имитируя тестовый горизонт.

### Бейзлайны
1. **Mean Baseline** — прогноз = среднее значение за 14 дней
2. **SeasonalNaive** — прогноз = значение той же недели (сезонность 7 дней)

### ML-модель: CatBoost

### DL-модель: LSTM
