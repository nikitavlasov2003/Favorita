# Favorita Grocery Sales Forecasting

## Описание

Решение задачи прогнозирования продаж для соревнования [Corporación Favorita Grocery Sales Forecasting](https://www.kaggle.com/competitions/favorita-grocery-sales-forecasting) на Kaggle.

**Метрика**: NWRMSLE (Normalized Weighted Root Mean Squared Logarithmic Error)

## Данные

Датасет расположен по пути `/kaggle/input/datasets/nikolasking/favorita/`.

Файлы:
| Файл | Описание |
|------|----------|
| `train.csv` | Продажи по дате, магазину, товару (2013–2017) |
| `test.csv` | 16-дневный тестовый горизонт (авг 2017) |
| `stores.csv` | Метаданные магазинов (город, штат, тип, кластер) |
| `items.csv` | Метаданные товаров (семейство, класс, скоропортящийся) |
| `transactions.csv` | Количество транзакций по дате и магазину |
| `oil.csv` | Цены на нефть (экзогенный признак) |
| `holidays_events.csv` | Праздники и события в Эквадоре |

---

## Описание подхода

### Постановка задачи
Предсказать ежедневные продажи (`unit_sales`) для каждой пары (магазин, товар) на 16 дней вперёд.

### Протокол валидации
Time-based split: последние 16 дней обучающей выборки (01–15 авг 2017) используются как локальная валидация, имитируя тестовый горизонт.

### Бейзлайны
1. **Naive** — прогноз = последнее наблюдение
2. **SeasonalNaive** — прогноз = значение той же недели (сезонность 7 дней)

### ML-модель: CatBoost

### DL-модель: LSTM
